# Professional Test Plan for CV Tailor App

این برنامه برای تست دقیق تک تک دکمه‌ها و تعاملات برنامه طراحی شده است تا اطمینان حاصل کنیم که فرانت‌اندمان به درستی به بک‌اندمان متصل است.

## بخش ۱: مدیریت سی‌وی (CV Manager)
**هدف**: بررسی امکان ایجاد و ذخیره یک پروفایل پایه.

- [ ] **بارگذاری فایل**: دکمه "Upload CV" را بزنید و یک فایل (PDF/Docx) آپلود کنید. آیا استخراج اطلاعات (Extraction) شروع می‌شود؟
- [ ] **ورود دستی**: تب "Enter Manually" را تست کنید. آیا فرم‌ها باز می‌شوند؟
- [ ] **ذخیره اطلاعات پایه**: نام، ایمیل و خلاصه را پر کنید و دکمه "Save CV" را بزنید. آیا پیام موفقیت ظاهر می‌شود؟
- [ ] **افزودن تجربه کاری**: دکمه "Add Experience" را بزنید. آیا ردیف جدید اضافه می‌شود؟ آیا می‌توانید آن را حذف کنید؟
- [ ] **افزودن مهارت**: بخش "Skills" را تست کنید. آیا تگ‌ها اضافه می‌شوند؟

## بخش ۲: ایجاد درخواست جدید (New Application)
**هدف**: بررسی فرآیند تولید محتوا توسط هوش مصنوعی.

- [ ] **انتخاب سی‌وی**: آیا سیستم سی‌وی ذخیره شده را شناسایی می‌کند؟
- [ ] **ورودی شغل**: عنوان شغلی و توضیحات (Job Description) را وارد کنید.
- [ ] **تنظیمات هوش مصنوعی**: انتخاب Provider (Gemini/Groq) و Model. آیا لیست مدل‌ها به درستی آپدیت می‌شود؟
- [ ] **تولید (Generate)**: دکمه "Generate Application" را بزنید. آیا لودینگ نمایش داده می‌شود؟ آیا به صفحه جزئیات منتقل می‌شوید؟

## بخش ۳: داشبورد و لیست‌ها
**هدف**: بررسی نمایش داده‌ها و ناوبری.

- [ ] **داشبورد**: آیا آمار (Total/Drafts/Completed) درست نمایش داده می‌شود؟
- [ ] **لیست درخواست‌ها**: به صفحه `/applications` بروید. آیا ردیف‌ها قابل کلیک هستند؟
- [ ] **فیلترها**: جستجو یا فیلتر وضعیت را تست کنید.

## بخش ۴: تنظیمات و پرامپت‌ها
**هدف**: بررسی بخش مدیریتی.

- [ ] **کلیدهای هوش مصنوعی**: به بخش تنظیمات بروید و یک کلید جدید اضافه کنید. آیا ذخیره می‌شود؟
- [ ] **مدیریت پرامپت**: آیا می‌توانید یک پرامپت را فعال/غیرفعال کنید؟

---
### نحوه اجرا:
ما این تست‌ها را مرحله به مرحله انجام می‌دهیم. بعد از هر مرحله، اگر دکمه‌ای کار نکرد، من کد آن را بررسی و بلافاصله اصلاح می‌کنم.

**بیایید از "بخش ۱: مدیریت سی‌وی" شروع کنیم. آیا آماده هستید؟**
